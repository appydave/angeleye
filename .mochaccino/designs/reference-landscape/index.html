Synthesis — Three Big Patterns for AngelEye

Consolidated from deep dives into Kibitz, Citadel, Claude Agent Teams UI, and RuFlo (2026-03-27)
01

Deterministic Classification Layer

Every system builds this independently. No LLM needed — pure event counting and regex.

  • Kibitz's event counting + drift heuristics
  • Citadel's circuit breaker thresholds
  • RuFlo's Q-Learning + MoE gating
  • Agent Teams UI's activity timing
→ AngelEye Enrichment Tier 1: zero LLM cost, fires on every hook event
02

Campaign = Chain-Story Data Model

AngelEye's chain-story IS a campaign. The schema is now clear.

  • Citadel's campaign file schema = data model
  • RuFlo's event sourcing = persistence pattern
  • Agent Teams UI's kanban-state.json = visualization state
  • Kibitz's pre-LLM assessment = automatic metadata
→ Combine all four into AngelEye's chain-story schema
03

Cross-Session Memory Without LLM

Compress each session into a structured brief for cheap cross-session intelligence.

  • Citadel: HANDOFF blocks (~150 words) + discovery relay (~500 tokens)
  • RuFlo: ReasoningBank retrieval (32% token reduction)
  • Kibitz: nothing (single-session — the gap AngelEye fills)
→ Structured briefs for cross-session intelligence at near-zero cost

AngelEye's Differentiation

Ambient intelligence — pattern mining, skill suggestion, anomaly detection (not just monitoring)
Cross-session threads — connecting related sessions into workspace narratives
Enrichment pipeline — three-tier detection (deterministic / partial / LLM)
BMAD story lifecycle — chain-story pipeline tracking across agent sessions
No control plane (yet) — observe and enrich, don't orchestrate. Job 5 is exploratory.

Key Patterns Across the Landscape

1. tmux + worktree isolation — dominant coordination pattern
2. Multi-CLI trending — Claude, Codex, Gemini as interchangeable backends
3. Space exploded Jan–Mar 2026 — most projects are weeks old
4a. Native Claude Code camp — skills/hooks (Agents, Citadel, Vibecosystem)
4b. External spawners — spawn CC processes (Shogun, Genie, ccswarm)
4c. Agent-neutral layers — runtime-agnostic (MCO, Overstory)
5. Observability emerging as sub-category — AngelEye is here, with richer ambient intelligence
