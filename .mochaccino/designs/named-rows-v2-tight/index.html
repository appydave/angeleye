What changed from v1

  • Session ID is hover-only. It's there when you need to copy it, invisible otherwise. Removes 8 characters of noise from every row.
  • Middot separator between name and project context — cleaner than a space, lighter than a badge.
  • Smaller badges (9px) — still readable, takes less horizontal space.
  • Grid layout instead of flexbox — dot, content, and right-cluster are always aligned across rows.
  • Tighter vertical padding (8px vs 10px) — fits more rows on screen without feeling cramped.
