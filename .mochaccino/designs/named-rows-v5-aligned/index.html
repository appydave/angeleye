What this fixes

  • Type badges line up vertically. BUILD, KNOWLEDGE, ORIENTATION are in a fixed column — your eye can scan straight down. This was the main problem with the current Observer.
  • When and Pulse also align. Time column and pulse column are fixed-width, so they stack cleanly too.
  • Same linen card style. Rounded borders, shadow-sm, hover shadow-md, same colour tokens, Bebas Neue for badges. This IS the current Observer — just with a grid backbone.
  • Named rows: amber border + bolder name + project context. That's it. No extra badges, no extra lines, no pennants. The name replaces the project as hero text, project slug appears dimmed after it.
  • Session ID stays visible (dimmed, after the name/project). Available for copy-click without hover tricks.
  • Full app shell included — sidebar, header, filters, focus panel — so you can see it in context, not isolated.
  • Column headers using Bebas Neue match the existing badge style. Lightweight guide for the grid.
