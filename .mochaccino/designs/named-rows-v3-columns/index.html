What changed from v1/v2

  • Fixed-width columns for when, pulse, star. These always align vertically across rows — your eye can scan the "pulse" column instantly to find active sessions.
  • Session ID removed entirely. It's a developer tool, not a user tool. Available on click/copy if needed, but not visible.
  • Badge stays in the session cell (inline with the name/project) rather than in its own column — it's metadata about the session, not a separate dimension.
  • Column headers make the grid structure explicit. Matches the "data table" mental model without feeling spreadsheet-heavy.
  • 44px minimum row height — tighter than v1 (52px). More sessions visible at once.
